相信各位玩家都曾經試過因為交易問題而導致受騙而求助無門，又苦苦找不到交易時的聊天記錄，而白白含冤受屈，現在《夢之仙境》將會為你打破困局，設立獨有的 Message Live 聊天系統，把所有聊天時的資料都記錄下來。


 

各位玩家只需點擊「社交」界面中的「好友」，開啟視窗後，從你的好友選單中，選擇你需要進行對話的好友，然後點擊右鍵，就會出現以下視窗，從畫面中大家可以看到「 Message Live 」，再次點擊「 Message Live 」就可以進行單獨且具有資料記錄的聊天視窗



↑快來點擊你的好友、交易目標進行有記錄的獨文聊天吧

 

在獨立聊天視窗裡，玩家不單只可以選擇更多的表情動作來代表你的情感，更可以進行各種邀請，如好友、組隊、家族等等。而更重要的是，在聊天畫面的左下角有一按鈕「聊天記錄」，只要玩家點擊聊天記錄，就可以查看當日與該玩家的聊天資料，資料內會詳述玩家的聊天經過、時間、過程。玩家更可透過「Message Live」看到你的好友的時裝，了解到他的生活品味！
